The nJord Platform offers some basic functions for manipulating the filesystem and reading/writing plain text files built into the core executable. The functions included in the basicfile module are:

  * [CopyFile](CopyFile.md)
  * [MoveFile](MoveFile.md)
  * [DeleteFile](DeleteFile.md)
  * [CreateDirectory](CreateDirectory.md)
  * [RemoveDirectory](RemoveDirectory.md)
  * [FileGetContents](FileGetContents.md)
  * [FileSetContents](FileSetContents.md)
  * [GetFileAttributes](GetFileAttributes.md)
  * [SetFileAttributes](SetFileAttributes.md)
